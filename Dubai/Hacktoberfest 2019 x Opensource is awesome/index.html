Business Impact

  • Flexibility and agility
  • Cost-effectiveness
  • Attract better talent
Include that me and Nodes convince clients to use open source.
  • Flexibility and agility - multiple ways of solving problems, not blocked because a particular capability
  • Speed - Improve Software on a User and Business Level, start quickly and inexpensively
  • Ability to start small - by using community versions(magento) you can start quick, test many solutions in short time and later scale up with commercial solution
  • Cost-effectiveness - professional support, faster involvement of new developers
  • Share maintenance cost - community involvement, well written documentation
  • Solid information security - by continous code reevaluation by many people(especially old code), problems could be identified faster and fixed
  • Attract better talent - most of professionals like to create projects, interact with others outside enterprise to solve problems
